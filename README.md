# ReSide

> Театрализованное воплощение оверинженерной микросервисной архитектуры.

ReSide - это экосистема небольших автономных агентов, называемых "репликами". Они реализуют различные (без)полезные функции и взаимодействуют с пользователями через CLI, UI или мессенджеры.
