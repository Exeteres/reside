# Changelog

## 0.1.14 - 2026-06-12

Добавлена команда /clear_context для очистки NLS-контекста пользователя у выбранной реплики.

## 0.1.13 - 2026-06-12

Обновлены инструкции поведения языкового интерфейса для пользователей Телеграмной Реплики.

## 0.1.12 - 2026-06-12

Телеграмная реплика обновила чтение секретов под типизированный формат.

## 0.1.11 - 2026-06-12

Уведомления, отправленные из контекста команды в обычном чате, снова отвечают на сообщение с вызовом команды.

## 0.1.10 - 2026-06-12

Привязка канала уведомлений к текущей теме Telegram теперь создает отсутствующую запись темы от имени Телеграмной Реплики.

## 0.1.9 - 2026-06-12

Добавлена команда /info, которая показывает канал уведомления и субъект-отправитель для сообщения, на которое отвечает команда.

## 0.1.8 - 2026-06-10

Команда /bind_notification_channel теперь принимает только имя канала и автоматически привязывает канал к теме Telegram, из которой она вызвана.

## 0.1.7 - 2026-06-10

Токены контекста взаимодействия Telegram теперь шифруются через общий crypto.encrypt/decrypt без отдельного Kubernetes-секрета.

## 0.1.6 - 2026-06-07

Добавлено хранение Telegram user id, username, first_name и second_name в отдельных ECID-полях пользователя с актуализацией на каждом запросе без лишней ротации ECID.

## 0.1.5 - 2026-06-07

Добавлена подстановка ECID в отправляемых уведомлениях и ответах NLS с безопасной расшифровкой и обобщённым русским представлением нестроковых значений.

## 0.1.4 - 2026-06-07

Идентификатор пользователя Telegram теперь хранится как RHID вместо открытого внешнего идентификатора.

## 0.1.3 - 2026-06-06

Исправлено обновление тега версии аватара: отсутствие аватара для реплики больше не приводит к ошибке, а специальная обработка Telegram удалена.

## 0.1.2 - 2026-06-06

Телеграмная реплика теперь игнорирует обновление version tag для самой себя и автоматически повышает добавленные в чат аватар-боты до администратора с минимальными правами.

## 0.1.1 - 2026-06-06

Добавлено корректное логирование ошибок в updateAvatarVersion и в серверных обработчиках Telegram.
