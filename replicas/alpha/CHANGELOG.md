# Changelog

## 0.2.4 - 2026-07-10

Исправлено ожидание загрузки реплик: операция теперь завершается ошибкой, если запрошенная версия не выше версии в кластере, а теги без версии заменяются новой версией.

## 0.2.3 - 2026-07-06

Убраны лишние префиксы из названий инструментов.

## 0.2.2 - 2026-07-03

Действия удаления реплики помечены для безопасного порядка выполнения.

## 0.2.1 - 2026-07-02

Теперь тег образа перезаписывается только при увеличении версии.

## 0.2.0 - 2026-06-22

Альфа Репликой была добавлена поддержка снятия регистрации реплики и удаления ее из кластера через Реплику-Убийцу.

## 0.1.3 - 2026-06-12

Обновлены инструкции поведения языкового интерфейса для пользователей Альфа Реплики.

## 0.1.2 - 2026-06-12

Исправлен лишний пустой перенос перед разделом «Изменения» в уведомлении об обновлении реплики.

## 0.1.1 - 2026-06-07

Добавлена базовая поддержка шифрования реплики через регистрацию EncryptionService и модель EncryptedContent.
