# Changelog

## 0.2.3 - 2026-07-03

Действия очистки прав помечены как обязательные перед удалением реплики.

## 0.2.2 - 2026-07-03

Добавлены недостающие статические разрешения для запуска системных реплик без перезаписи их описаний.

## 0.2.1 - 2026-07-03

Добавлены статические разрешения для запуска Телеграмной Реплики без запроса подтверждения.

## 0.2.0 - 2026-06-22

Авторизационной Репликой была добавлена поддержка удаления биндингов и подтверждающих через Реплику-Убийцу.

## 0.1.3 - 2026-06-12

Обновлены инструкции поведения языкового интерфейса для пользователей Авторизационной Реплики.

## 0.1.2 - 2026-06-07

Добавлена базовая поддержка шифрования реплики через регистрацию EncryptionService и модель EncryptedContent.

## 0.1.1 - 2026-06-07

Добавлено системное разрешение для передачи зашифрованного содержимого между репликами.
