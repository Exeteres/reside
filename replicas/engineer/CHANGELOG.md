# Changelog

## 0.3.1 - 2026-07-03

Добавлена подсказка использовать git revert при удалении реплик из исходного кода.

## 0.3.0 - 2026-07-02

Добавлен инструмент создания временных баз данных PostgreSQL для выполнения инженерных задач.

## 0.2.0 - 2026-06-22

Инженерной Репликой была добавлена поддержка создания плановой задачи на удаление исходного кода через Реплику-Убийцу.

## 0.1.25 - 2026-06-13

В Инженерной Реплике уточнены инструкции выполнения исследовательских задач и обработки нерелевантных сбоев широких проверок.

## 0.1.24 - 2026-06-13

В Инженерной Реплике добавлен короткий путь выполнения задач из языкового интерфейса без планирования и создания issue.

## 0.1.23 - 2026-06-13

Инженерной Реплике расширение разрешенных системных инструментов агента переведено на общие настройки по умолчанию.
Инженерной Реплике добавлен валидирующий инструмент создания коммитов и уточнены инструкции по редактированию файлов агентом.

## 0.1.22 - 2026-06-13

Инженерной Реплике добавлена ссылка на общие правила изменений для версий, changelog и коммитов.

## 0.1.21 - 2026-06-12

Обновлены инструкции по созданию новых реплик из примерного шаблона.

## 0.1.20 - 2026-06-12

Исправлена доставка изменений Инженерной Реплики через единый инструмент, улучшено чтение ошибок CI и добавлена подсказка использовать скрипт создания новых реплик.

## 0.1.19 - 2026-06-12

Для Инженерной Реплики увеличено ожидание активности языкового движка при планировании и выполнении задач.

## 0.1.18 - 2026-06-12

В Инженерной Реплике исправлена отмена планирования и выполнения задач с закрытием issue и топика.

## 0.1.17 - 2026-06-12

Инженерной Реплике добавлена инструкция запускать проектные инструменты через devenv.

## 0.1.16 - 2026-06-12

Инженерной Реплике разрешены встроенные инструменты агентной разработки Copilot.

## 0.1.15 - 2026-06-12

Добавлена повторная попытка для упавших итераций Инженерной Реплики с переоткрытием топика и issue.

## 0.1.14 - 2026-06-12

Запуск планирования и выполнения инженерных задач больше не блокируется ожиданием ответа на прогресс-уведомление.

## 0.1.13 - 2026-06-12

Топик инженерной задачи теперь закрывается после терминального завершения выполнения.

## 0.1.12 - 2026-06-12

Исправлено экранирование потокового прогресса инженерных задач и падение при ошибках обновления уведомлений.

## 0.1.11 - 2026-06-12

Уведомление о созданной инженерной задаче переведено на кнопку со ссылкой без текстового тела.

## 0.1.10 - 2026-06-12

Переработан промпт планирования инженерных задач для фиксации контекста и требований вместо пошагового плана.

## 0.1.9 - 2026-06-12

Исправлено потоковое обновление прогресса инженерных задач без накопления устаревших фрагментов.

## 0.1.8 - 2026-06-12

Исправлено зависание подготовки инженерной задачи при создании начального уведомления прогресса.

## 0.1.7 - 2026-06-12

Исправлена подготовка инженерных задач при неструктурированном ответе модели и добавлено уведомление об ошибке создания задачи.

## 0.1.5 - 2026-06-12

Переработан жизненный цикл инженерных задач через подготовку топика, отдельный долгоживущий workflow и обновление прогресса в уведомлениях.
