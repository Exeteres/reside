# Changelog

## 0.1.15 - 2026-06-12

Добавлена повторная попытка для упавших итераций Инженерной Реплики с переоткрытием топика и issue.

## 0.1.14 - 2026-06-12

Запуск планирования и выполнения инженерных задач больше не блокируется ожиданием ответа на прогресс-уведомление.

## 0.1.13 - 2026-06-12

Топик инженерной задачи теперь закрывается после терминального завершения выполнения.

## 0.1.12 - 2026-06-12

Исправлено экранирование потокового прогресса инженерных задач и падение при ошибках обновления уведомлений.

## 0.1.11 - 2026-06-12

Уведомление о созданной инженерной задаче переведено на кнопку со ссылкой без текстового тела.

## 0.1.10 - 2026-06-12

Переработан промпт планирования инженерных задач для фиксации контекста и требований вместо пошагового плана.

## 0.1.9 - 2026-06-12

Исправлено потоковое обновление прогресса инженерных задач без накопления устаревших фрагментов.

## 0.1.8 - 2026-06-12

Исправлено зависание подготовки инженерной задачи при создании начального уведомления прогресса.

## 0.1.7 - 2026-06-12

Изменена подготовка инженерных задач на ручную проверку JSON-ответа модели при генерации названия.

## 0.1.6 - 2026-06-12

Исправлена подготовка инженерных задач при неструктурированном ответе модели и добавлено уведомление об ошибке создания задачи.

## 0.1.5 - 2026-06-12

Переработан жизненный цикл инженерных задач через подготовку топика, отдельный долгоживущий workflow и обновление прогресса в уведомлениях.

## 0.1.4 - 2026-06-12

Исправлена обработка создания задач через языковой интерфейс Инженерной Реплики без контекста взаимодействия.

## 0.1.3 - 2026-06-12

Обновлены инструкции языкового интерфейса Инженерной Реплики для планирования задач.

## 0.1.2 - 2026-06-12

Реплика инженера теперь использует общий LLM-секрет для языкового движка.

## 0.1.1 - 2026-06-07

Добавлена базовая поддержка шифрования реплики через регистрацию EncryptionService и модель EncryptedContent.
