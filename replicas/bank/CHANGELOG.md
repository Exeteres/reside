# Changelog

## 0.2.5 - 2026-07-05

Чтение баланса и истории через API теперь ограничено текущим субъектом, комментарии хранятся зашифрованными, а языковая модель получает суммы и комментарии истории только в виде ECID.

## 0.2.4 - 2026-07-05

Команды больше не повторяются при ошибках ввода и недостатке средств.

## 0.2.3 - 2026-07-05

Ошибки банковских команд теперь используют доменные сообщения, а знак суммы в истории показывается без лишнего пробела.

## 0.2.2 - 2026-07-05

В сообщении о выпуске средств теперь показывается идентификатор субъекта, а ошибки команд отправляются пользователю.

## 0.2.1 - 2026-07-05

Изменен формат истории транзакций.

## 0.2.0 - 2026-07-05

Добавлена выдача приветственных средств новым телеграмным счетам.

## 0.1.1 - 2026-07-05

Исправлена выдача средств репликам при записи зашифрованного баланса.

## 0.1.0 - 2026-07-05

Создана начальная версия Банковская реплика.
