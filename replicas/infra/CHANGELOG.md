# Changelog

## 0.4.0 - 2026-07-11

Добавлено создание временных баз данных из копии существующей базы.

## 0.3.4 - 2026-07-05

Удаление базы данных теперь также отключает ее от Mathesar.

## 0.3.3 - 2026-07-04

Исправлена отправка действия удаления базы данных Реплике-Убийце.

## 0.3.2 - 2026-07-04

Операции удаления инфраструктурных ресурсов больше не пропадают до обновления статуса.

## 0.3.1 - 2026-07-03

Действия очистки инфраструктуры помечены как обязательные перед удалением реплики.

## 0.3.0 - 2026-07-02

Добавлено создание временных баз данных PostgreSQL с автоматическим удалением через 24 часа.

## 0.2.0 - 2026-06-22

Инфраструктурной Репликой была добавлена поддержка удаления баз данных, Temporal-неймспейсов и шлюзов через Реплику-Убийцу.

## 0.1.4 - 2026-06-13

Исправлена отправка журналов подов Инфраструктурной Реплики из SigNoz k8s-infra collector через корректный OTLP HTTP endpoint.

## 0.1.3 - 2026-06-12

Обновлены инструкции поведения языкового интерфейса для пользователей Инфраструктурной Реплики.

## 0.1.2 - 2026-06-07

Добавлена базовая поддержка шифрования реплики через регистрацию EncryptionService и модель EncryptedContent.

## 0.1.1 - 2026-06-07

Добавлен сервис выдачи параметров Vault для реплик.
