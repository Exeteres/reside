## Команды

Секретарь Пользовательской Реплики поддерживает следующие команды:

| Команда    | Описание                                                                                              |
| ---------- | ----------------------------------------------------------------------------------------------------- |
| `/profile` | Выводит информацию о профиле пользователя, включая его ID в системе и список выданных ему разрешений. |
